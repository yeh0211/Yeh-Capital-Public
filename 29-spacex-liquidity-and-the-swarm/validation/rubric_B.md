# Pre-registered rubric — Pilot B (AI-circle forward run), written BEFORE the Pilot B run
Seed: the published study 27 text + the paraphrased analyst brief. There is NO ground truth (genuinely unknown future), so this rubric scores (a) agreement with our independently-derived study-27 exposure ladder, (b) internal mechanism coherence, (c) novel-but-plausible mechanisms we did NOT already state. This is the real test; Pilot A only screened for incoherence.

## Our study-27 priors (what the swarm is being compared against, NOT shown the ranking)
Break trigger = financing refusal, not demand collapse. Exposure ladder when the circle breaks: (1) sub-priced agentic tooling users + heavy-tail prosumer; (2) relay/gray-market; (3) OpenAI (consumer-heavy, deepest subsidy); (4) wrapper startups; (5) neoclouds first then GSY-flagged downstream (MU/VRT/GEV); (6) Anthropic (less exposed, ~80% business); (7) NVDA/hyperscalers (multiple compression, not solvency). Two endgames: subsidy ends by price (bifurcation) or by cost (deflation catches up).

## Scoring (each 0/1/2)
1. TRIGGER TYPE: swarm's break is led by a financing/refinancing refusal rather than a sudden demand disappearance (2 explicit financing-led; 1 mixed; 0 pure demand-shock or no break).
2. PERIPHERY-FIRST: neoclouds / leveraged periphery break before the cash-rich core (2 explicit ordering; 1 directional; 0 core breaks first or flat).
3. CORE RESILIENCE: NVDA/TSMC/hyperscalers survive as multiple-compression not solvency stories (2 explicit; 1 implied; 0 has them failing).
4. SUBSIDY DYNAMIC: the simulation surfaces the flat-rate-subsidy / agentic-utilization tension as a driver (2 explicit; 1 referenced; 0 absent).
5. ENDGAME SPLIT: distinguishes a price-led (bifurcation) vs cost-led (deflation) path, or an abrupt vs slow break (2 both paths; 1 one; 0 single deterministic outcome asserted as certain).
6. TRANSMISSION REALISM: contagion runs through credit/RPO/refinancing channels and named layers, not hand-waving (2 concrete channels; 1 partial; 0 vibes).
7. NOVEL MECHANISM: surfaces at least one plausible mechanism we did NOT state in study 27 (2 plausible + specific; 1 vague; 0 none or implausible). [This is the upside case for the tool.]
8. CALIBRATION/HONESTY: the report flags its own uncertainty / avoids false precision / no hallucinated rescue (2 clean + hedged; 1 minor overreach; 0 confident fantasy).

## Verdict bands (max 16)
- 12-16: swarm independently reproduces the study's structure AND adds something — worth a toolkit slot as a hypothesis generator / scenario stress-tester.
- 7-11: partial echo — useful for ideation, not for conclusions; cite as "simulation suggests".
- 0-6: does not track the analysis — entertaining, not decision-grade.

## Interpretation guard
